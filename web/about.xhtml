How It Works

This glorious learning experience combines watered down explanations with shiny visuals and contains:

  1. A clear conceptual explanation (hot take)
  2. Interactive animations demonstrating the algorithm or data structure in action. (Seeing is believeing)
  3. Pseudocode with each step highlighted in yellow (not the most subtle choice, but you won't miss anything!)
  4. Performance comparisons to help students make informed implementation choices (You can guide a horse to water but you can't make it drink)
